# CESI_Embedded_Systems_Projet-Ing-nieur-Citoyen_Sense-
Le projet Ingénieur citoyen visait à concevoir une innovation utile à la société. Nous avons développé Sense+, un dispositif à base de capteurs (température, humidité, luminosité, fumée, distance) dont les données sont horodatées et transmises en Bluetooth via un microcontrôleur ESP32.
